## 月30万円を稼ぐための必要ロット数

USD/JPY: **1ロット（10万通貨）≒ 1,000円/pip**

```
月平均獲得: 86 pips/月
目標: 300,000円/月

必要ロット = 300,000 ÷ (86 × 1,000) ≈ 3.5ロット
```

**→ 約3〜4ロット（30〜40万通貨）**
