## 現在の設定

| 項目 | 設定値 |
|---|---|
| 通貨ペア | USD/JPY |
| トレンド判定 | **MA20傾きベース**（10本前のMAと比較、最小3pips差） |
| 判定軸 | **H4 + H1 の2軸一致** |
| エントリー価格 | H4スウィングポイント付近（SLバッファ5pips） |
| SL | H4スウィングポイントの反対側5pips |
| TP | SL × 2（R/R比 1:2固定） |
| proximityPips | ≤ 20pips（接近通知の閾値） |
| SL制限 | 15〜80pips（ノイズ・過大リスク除外） |
| M15フィルター | あり（M15が逆方向の場合スキップ） |

## 注意事項

- バックテストは **USD/JPY** のみ。他通貨ペアは未検証
- 「1ポジションずつ」（前ポジション決済後に次エントリー）でシミュレーション
- スプレッド・スリッページは考慮していない
- 過去データが将来の結果を保証するものではない
